**Core Rules Requirements (pp. 1747-1829):**
- 3 rolls on Military Weapon Table
- 3 rolls on Low-tech Weapon Table  
- 1 roll on Gear Table
- 1 roll on Gadget Table

- Equipment assignment and transfer
- Carrying capacity limits

### **5. CREW STARTING RESOURCES ❌**
**Core Rules Requirements (Background Table pp. 1520-1733):**

- Story Points from motivations
- Quest Rumors from classes
- Patrons from special backgrounds

### **6. SHIP COMPONENTS SYSTEM ❌**
**Core Rules Requirements (pp. 4876-5035):**

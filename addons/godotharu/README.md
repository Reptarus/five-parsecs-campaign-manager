# GodotHaru (Asset Library package)

## Installation

1. Extract this archive at the root of your Godot project.
2. Make sure this folder exists: `res://addons/godotharu/`.
3. Reopen the project in Godot 4.4+.
4. Use exposed classes such as `PDF_DOC`, `PDF_PAGE`, and `PDF_FONT`.

This package contains Linux and Windows prebuilt binaries.
